
Here are all the libraries used in the code:

yfinance - For fetching financial data.
pandas - For data manipulation and analysis.
tkinter - For creating the GUI.
tkinter.ttk - For using themed widgets in the GUI.
tkinter.messagebox - For displaying message boxes in the GUI.
tkinter.scrolledtext - For creating a scrollable text area in the GUI.
difflib - For finding close matches to strings.


MAKE SURE TO INSTALL THESE LIBRARIES BEFORE HAND, BEFORE YOU RUN THE PROGRAM
